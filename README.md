## 🎯 如何使用Agents

### 方法1: 直接调用（推荐）

在WorkBuddy对话中直接输入：

```
"使用[Agent名称]执行任务：[你的需求]"
```

**示例**：
```
"使用Reveal Specialist启动项目：我需要解决鞋服零售门店的用户选择困难症"
```

### 方法2: 查看文档

每个Agent都包含完整的文档：

1. **快速安装**: `[Agent目录]/INSTALL.md`
2. **用户指南**: `[Agent目录]/PUBLIC_README.md`
3. **完整文档**: `[Agent目录]/README.md`
4. **Agent定义**: `[Agent目录]/AGENT.md`
