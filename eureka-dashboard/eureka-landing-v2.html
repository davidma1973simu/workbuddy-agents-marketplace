❓ 常见问题

你可能还想知道

Eureka 真的完全免费吗？以后会收费吗？⌄
完全免费，没有付费功能，没有付费计划。AI 辅助功能需要你自己提供 API Key，但 Eureka 本身永久免费。
我的项目数据安全吗？会上传到服务器吗？⌄
所有数据都存储在你的浏览器 localStorage 中，不会上传到任何服务器。建议定期用 JSON 导出功能备份。
我不懂设计思维，也能用好 Eureka 吗？⌄
完全可以。每个字段都有清晰的填写说明，AI 也会帮你生成内容。即使是第一次接触设计思维，也能顺畅完成整个流程。
AI 辅助功能如何使用？需要额外付费吗？⌄
需要你自己提供 API Key，支持 OpenAI / DeepSeek / 通义千问等兼容接口。DeepSeek 的调用成本极低，通常几分钱就能完成一次分析。Eureka 本身零费用。
换了电脑/浏览器，数据还在吗？⌄
数据存在本地浏览器，换设备后不会自动同步。建议用"导出JSON"功能定期备份，新设备通过"导入JSON"恢复即可。
可以用于企业团队或培训课程吗？⌄
完全可以。许多培训师和企业创新团队已在使用 Eureka 沉淀工作坊成果。如需定制化合作或工作坊设计支持，欢迎联系我们。
