## 五、线上线下整合用户旅程

### 核心设计原则
> **实体是入口，数字是引擎，人是中心。**
> 盒子打开的那一刻是最强情绪峰值，数字工具让这个峰值延伸成持续的成长旅程。

### 四类用户路径
| 用户类型 | 首次接触点 | 核心旅程 | 价值兑现 |
|---|---|---|---|
| A 个人探索者 | 小红书/B站内容 | 免费Learn体验 | 购买盒子/升级 |

### 实体盒子内容设计（与数字工具对应）
| 盒子内容 | 对应数字模块 | 使用场景 |
|---|---|---|
| RISE 总览画布（A2×1） | 整体框架导航 | 贴墙/桌面展开，全局视角 |
| Reveal 画布（A3×1） | T1-T6 任务 | 用户调研/问题定义工作坊 |
| Inspire 画布（A3×1） | T7-T11 任务 | 创意发散/最佳创意筛选 |
| Shape 画布（A3×1） | T12-T15 任务 | 方案整合/故事板 |
| Exam 画布（A3×1） | T16-T20 任务 | 测试计划/验证闭环 |
| 用户卡片（20张） | Persona Lab | 用户角色快速构建 |
| 洞察卡片（20张） | T5 POV 工具 | 问题重构/观点卡 |
| 创意卡片（30张） | T8-T9 发散工具 | 头脑风暴触发 |
| 激活码卡片（1张） | 数字平台激活 | 绑定线上账号 |
| 使用手册（8页） | — | 首次使用引导 |

> **关键设计**：激活码绑定后，线下画布每个区域都有**二维码**，扫一扫直接跳转到对应的数字任务页面。线上/线下无缝切换。

```
 │          │         │            │            │

                     讲师版工具    第一笔收入   年度更新
```

**讲师核心价值主张**：
- 你来了解方法论 → Eureka 给你交付工具
- 你带客户 → Eureka 给你实体物料 + 数字后台
- 你赚钱 → Eureka 收授权费 + 盒子批发差价
- **三方共赢：讲师赚服务费，学员有完整体验，David 有被动收入**

### 五个关键情绪时刻
| 时刻 | 触发 | 设计要点 |
|---|---|---|
| **开箱时刻** | 收到实体盒子 | 精美包装+激活码+首次引导卡，情绪峰值设计 |

| **完成时刻** | 画布填满+PDF导出 | 实体拍照+数字成果，双重可分享 |
| **传播时刻** | 发社交媒体 | Eureka 品牌水印+用户故事，有机增长 |

## 七、系统可持续性设计

### 关键风险与应对
| 风险 | 应对策略 |
|---|---|
| 单次销售，无复购 | 订阅制 + 讲师授权 = 持续可预期收入 |
| 竞品复制方法论 | 实体盒子 + 认证体系建立品牌壁垒，方法论易复制，生态不易复制 |
| 个人精力瓶颈 | 讲师网络作为杠杆，David 专注产品和IP输出 |
| 技术债（localStorage） | Phase 2 接入腾讯云开发（TCB），云同步代码已写好待激活 |
| 线下物料成本 | 小批量印刷起步，验证需求后规模化，讲师批发分担库存压力 |

### 收入预测模型（保守估计）
```
Phase 1（月3）：
  - 盒子预售：100套 × ¥399 = ¥39,900

Phase 2（月6，月度）：
  - 盒子持续销售：30套/月 × ¥399 = ¥11,970

  - 企业项目：1单/月 × ¥10,000 = ¥10,000
  - MRR 合计：约 ¥28,630

Phase 3（月12，月度）：
  - 盒子+订阅：¥20,000
```
