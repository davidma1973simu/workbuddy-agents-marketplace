⚙️ AI 配置与调用指南

Eureka 的 AI 智能体功能需要配置 API Key 才能使用。支持 OpenAI、DeepSeek、通义千问等主流大模型服务商，也可通过 WorkBuddy 直接调用已配置的技能和智能体。

🔑
API 配置
Dashboard 内配置
进入 Eureka Dashboard 后，点击右上角「⚙️ AI 设置」按钮，填写 API Key、选择服务商、设置模型名称即可。
支持服务商：OpenAI · DeepSeek · 通义千问 · 自定义接口
🛰️
WorkBuddy 集成
直接调用 Skills
在 WorkBuddy 中已配置的 Eureka Skills 可直接调用，无需重复设置 API Key。支持 pov_builder、HMW_Reframer、NCO_Crosslinker 等 10+ 个创新技能。
调用方式：在 WorkBuddy 聊天框直接输入技能名称或描述需求
🤖
Sub-Agent
四阶段专家智能体
Eureka 提供 Reveal / Inspire / Shape / Exam 四个 Specialist Agent，可在 Dashboard 各阶段编辑器中点击「✨ AI 助手」按钮唤醒，自动识别当前上下文并生成结构化产出。
使用场景：POV 构建 · HMW 重构 · 创意发散 · 概念打磨
