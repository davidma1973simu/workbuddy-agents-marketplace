## 🚀 立即使用

### 方法1: 直接调用（推荐）

在WorkBuddy对话中直接输入：

```
"使用Reveal Specialist启动项目：[你的需求]"
```

**示例**:
```
"使用Reveal Specialist启动项目：我需要解决鞋服零售门店的用户选择困难症"
```

### 方法2: 分步调用

```
第一步："使用build_project_brief构建项目简报"
第二步："使用roleplay_stakeholder分析利益相关方"
第三步："使用find_insight挖掘FIND洞察"
第四步："使用pov_builder构建用户POV"
```

## 🎯 核心功能

- 🔍 **用户研究** - 访谈、观察、数据分析
- 💡 **洞察挖掘** - FIND模型（事实→解读→需求→洞察）
- 🤝 **利益相关方** - Buy Features方法
- 📄 **需求整合** - 生成用户POV

---

## 📁 输出内容

使用后，你将获得6个完整文件：

1. **00_build_project_brief.json** - 项目简报
2. **02_find_insight.json** - FIND洞察
3. **01_pov_builder.json** - 用户POV
4. **03_roleplay_stakeholder.json** - 利益相关方分析
5. **pov_card.html** - POV卡片可视化
6. **stakeholder_requirements_list.txt** - 需求优先列表
