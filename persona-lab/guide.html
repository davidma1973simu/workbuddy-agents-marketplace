产品定位
Persona Lab 是什么？

Persona Lab 是一款专为创新设计流程打造的虚拟用户角色管理平台，通过 AI 生成与结构化模板双引擎，帮助团队在项目启动阶段快速建立对目标用户的共同认知。

💡 为什么需要虚拟人？

在传统设计过程中，团队常常基于模糊的「典型用户」做决策——这导致方向偏差、需求误判和大量返工。虚拟人（Persona）将真实调研数据提炼为有名字、有故事、有行为模式的具体角色，让团队在每一个设计决策中都能回答：「这个用户会怎么想？」

使用指南
三步开始你的第一个角色组

无需注册、无需安装，打开即用。以下是从零到角色组的完整流程。

✨

AI 生成模式

输入项目主题，AI 理解语义后自动生成多类角色。支持 OpenAI / DeepSeek / 通义千问等大模型接入。

推荐
🗂

市场浏览模式

浏览 33+ 个预置角色，按行业和类型筛选，找到合适的角色直接添加到角色组。

📦

角色组管理

查看和管理所有已保存的角色组，支持重命名、导出为 Markdown / PDF 格式。

1

描述你的项目主题

在「✨ AI 生成」模式下，在顶部输入框中用自然语言描述你的项目。无需格式要求，说清楚你在做什么就好。

示例输入：
「面向老年人的智慧医疗健康管理 App」
「新能源汽车 4S 店的数字化服务体验优化」
「面向 Z 世代的在线职业培训平台」
2

（可选）精细化标签筛选

左侧标签面板提供行业、场景、主题三维度标签。勾选标签可以进一步缩小生成范围，获得更精准的结果。AI 生成模式下标签用于引导语义匹配；市场浏览模式下标签用于过滤预置角色。

标签维度说明：
行业标签（14个）：科技、医疗、金融、教育、汽车…
场景标签（6个）：产品研发、服务设计、体验优化、数字化转型…
主题标签（6个）：老龄化、认知负荷、决策辅助、可及性…
3

生成并加入角色组

点击「✨ 生成虚拟人」，系统将生成一批角色卡片。查看每张卡片的详情（点击展开抽屉），将满意的角色点击「+ 加入角色组」。

小技巧：可以多次生成不同主题或配置不同标签，将多批生成的角色混合加入同一个角色组，形成更多元的人物图谱。
4

命名并保存角色组

右侧角色托盘（Group Tray）显示当前已添加的角色。填入角色组名称，点击「💾 保存角色组」。保存后可在「📦 我的角色组」中随时查看。

5

导出或同步到 Eureka

在「我的角色组」中，选择角色组后可以：

  • 📄 导出为 Markdown —— 嵌入 Notion / Confluence / 飞书文档
  • 🖨 导出为 PDF —— 打印或插入 PPT 提案
  • 🔗 同步到 Eureka —— 一键将角色导入创新项目库的 Reveal 阶段
AI 大模型接入
让 AI 为你定制专属角色

Persona Lab 内置语义模板引擎，即使不配置 AI 也能正常使用。接入大模型后，生成结果将更加个性化和贴合项目上下文。

🤖 支持的 AI 服务

OpenAI (GPT-4o / GPT-4 / GPT-3.5)、 DeepSeek（deepseek-chat）、 通义千问（qwen-plus）、 Azure OpenAI 及其他兼容 OpenAI Chat Completions 格式的接口均可接入。

⚙️

打开 AI 设置面板

点击顶部导航栏右侧的「⚙️ AI 设置」按钮，打开配置面板。

🔑

填入 API Key 和模型

选择 API 提供商，填入 API Key 和模型名称。如使用国内兼容接口，切换为「兼容接口」模式并填入自定义端点地址。配置信息仅存储在本地浏览器，不会上传至任何服务器。

✅

生成时自动切换至 AI 模式

保存配置后，生成时将优先调用 AI 接口。如 AI 接口调用失败，系统将自动降级至内置模板引擎，不影响正常使用。角色卡片会标注「✨ AI 生成」或「📋 模板引擎」来源。

常见问题
FAQ
Persona Lab 是否需要登录注册？ ▾
不需要。Persona Lab 是纯前端应用，所有数据存储在你的浏览器本地（localStorage）。无需注册、无需服务器，打开即用。

⚠️ 注意：清除浏览器数据会导致角色组丢失，建议定期导出为 Markdown 或 PDF 备份。
我的 API Key 安全吗？会被上传吗？ ▾
绝对安全。API Key 仅存储在你的浏览器本地（localStorage key: persona_lab_ai_config_v1），Persona Lab 不与任何后端服务器通信，你的 Key 不会被上传或共享。

AI 请求是从你的浏览器直接发送至 AI 服务商（OpenAI 等），不经过任何中间服务器。
不配置 AI 能正常使用吗？ ▾
完全可以。Persona Lab 内置了覆盖 14 个行业的语义模板引擎，无需 AI 即可生成有意义的角色。AI 接入是可选的增强功能，让生成结果更加个性化，但并非必需。
如何把角色插入到我们的 PPT 或文档中？ ▾
两种方式：

1. PDF 导出：在「我的角色组」中选择角色组，点击「🖨 导出 PDF」，调用浏览器打印功能，保存为 PDF 文件后插入 PPT 或直接作为附件发送。

2. Markdown 导出：导出后可粘贴至 Notion、Confluence、飞书文档等工具，或在 Markdown 编辑器中进一步编辑。
生成的角色可以修改吗？ ▾
目前角色卡片支持查看详情（点击展开抽屉），直接编辑字段的功能在规划中。当前建议：将角色添加到角色组后，通过 Markdown 导出在外部文档工具中进行个性化修改。
Persona Lab 可以在手机上使用吗？ ▾
可以访问，但界面针对桌面端优化，手机端体验有限。我们计划推出 PWA 版本以改善移动端体验，后续也在评估微信小程序 Viewer 版本（角色卡片只读展示）。
33 个预置角色能满足我的行业需求吗？ ▾
预置角色覆盖 14 个主要行业域，主要用于快速参考和启发。如果找不到完全匹配的角色，建议使用「AI 生成」模式，输入你的具体项目描述，语义引擎会生成更贴合的定制角色。如需在团队间共享自定义行业模板，欢迎通过产品反馈告知我们。
