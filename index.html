📋 如何使用 Agents

🎯 在 Eureka 项目库中一键调用

每个 Agent 对应 Eureka 的一个创新阶段，直接在项目编辑器中调用，无需手动输入命令。

🔍 洞察官 · Reveal 阶段
打开项目 → 编辑 → 切到「Reveal」Tab → 点击右上角「🤖 AI助手」→ 选择「FIND洞察」或「POV构建」等能力
💡 启发师 · Inspire 阶段
打开项目 → 编辑 → 切到「Inspire」Tab → 点击右上角「🤖 AI助手」→ 选择「HMW机遇」或「创意发散」等能力
🎨 架构师 · Shape 阶段
打开项目 → 编辑 → 切到「Shape」Tab → 点击右上角「🤖 AI助手」→ 选择「方案拷问」或「体验故事」等能力
✅ 验证官 · Exam 阶段
打开项目 → 编辑 → 切到「Exam」Tab → 点击右上角「🤖 AI助手」→ 选择「AHA评估」或「电梯呈现」等能力
💡 提示：AI 助手会自动读取当前项目的已有数据作为上下文，生成结果可直接回填到项目表单。首次使用需在 Dashboard 中配置 AI 接口。

🚀 或在 WorkBuddy 对话中直接调用

如果你更习惯对话式交互，也可以在 WorkBuddy 中直接输入命令调用 Agent：

使用 Reveal Specialist 启动项目：我需要解决鞋服零售门店的用户选择困难症
使用 Inspire Specialist 发散创意：用户POV：职场新人在远程协作中感到孤立
